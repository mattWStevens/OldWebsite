ARM Assembly Language Emulator

This is an Android application that emulates a code editor for ARM Assembly Language. It enables users select the line of the text editor that they desire and then they are prompted for what type of operation they would perform. The user is then able to select operation type and paramters, such as constants and register numbers. The editor is then updated with the corresponding operation code. Upon executing the play button, the user is able to visually see how the binary numbers being stored in the computer's memory or register banks are altered.

Visit Repository

Hangman Web Browser Game

This is a web browser game variation of the classic game of hangman. It comes with a slight twist, however. The user is only notified of the correct letters that they guess, not the location of the correct letters.
